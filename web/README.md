React + TypeScript + Vite web app for **Style Savvy**.

## Setup

```bash
npm install
cp .env.example .env
```

Edit `.env` and set `VITE_API_BASE_URL` to your API URL (e.g. `http://localhost:8000` for local dev).

## Run

```bash
npm run dev
```

Open http://localhost:5173. Ensure the API is running from the project root (see root [README](../README.md)).

## Build

```bash
npm run build
```

Output is in `dist/`. Deploy to Vercel, Netlify, or any static host.

For full project setup, database, and deployment, see the [root README](../README.md).
